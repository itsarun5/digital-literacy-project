# Project Overview
This repository contains my coursework and practical tasks as part of the Student Digital Ambassador initiative. The goal of the project is to promote digital literacy, safe online practices, and professional communication among college students. Each task focuses on building awareness, applying digital tools, and creating resources that can be shared with peers.

# Module Summaries
- Task 1 – Presentation
Designed an infographic on Digital Literacy using Canva. Covers topics like safe internet practices, useful digital tools, and professional online presence.
- Task 2 – Portfolio Setup
Created and updated accounts on GitHub, LinkedIn, and Kaggle. Added a GitHub profile README, filled in education details on LinkedIn, and set up Kaggle for data science practice.
- Task 3 – Platforms
Built a Digital Literacy Awareness Quiz using Google Forms, linked responses to Google Sheets, and added screenshots to the repository.
- Task 4 – Email Etiquette & Social Media
Drafted professional emails (to a professor and an internship coordinator) and prepared a checklist of social media do’s and don’ts for responsible use.
- Task 5 – Cybercrime Awareness
Wrote a case study on phishing, created a prevention checklist with UPI safety tips, and documented reporting channels like the National Cyber Crime Portal and Helpline 1930
